# Anti-Gravity Portfolio

A futuristic, space-themed personal portfolio website built with React, Tailwind CSS, and Framer Motion.

## Features

- **Anti-Gravity Design**: Elements float and drift continuously.
- **Glassmorphism**: Premium glass cards and interfaces.
- **Responsive**: Fully optimized for mobile and desktop.
- **Performance**: Smooth 60fps animations.

## Tech Stack

- **Framework**: React + Vite
- **Styling**: Tailwind CSS v3
- **Animations**: Framer Motion
- **Icons**: Lucide React

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Build for production:
    ```bash
    npm run build
    ```

## Customization

-   **Content**: Edit files in `src/sections/` to update text and projects.
-   **Theme**: Modify `tailwind.config.js` to change colors/fonts.
